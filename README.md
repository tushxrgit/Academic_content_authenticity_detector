# AI-Powered Academic Content Authenticity Detector

## 📌 Project Overview
This project is designed to detect whether academic content (assignments, essays, reports) is written by a human or generated using AI. It helps maintain academic honesty by analyzing writing patterns and content structure.

---

## 🚀 Features
- Upload PDF assignments  
- Detect AI-generated vs human-written content  
- Provides AI probability score  
- Supports both typed and handwritten documents (OCR)  
- Simple and user-friendly interface  

---

## 🧠 Working Principle
The system analyzes the content using:
- Semantic similarity  
- Vector embeddings  
- Writing pattern analysis  
- Perplexity-based evaluation  

It compares the input text with known patterns of AI and human writing and generates a probability score.

---

## 🛠️ Tech Stack
- Python  
- Sentence Transformers  
- GPT-2 (Transformers)  
- FastAPI  
- PyMuPDF  
- Pytesseract (OCR)
